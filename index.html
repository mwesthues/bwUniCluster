bwUniCluster Introduction

This repository is meant to aid you with learning how to navigate on the bwUniCluster and provides you with some basic material for running your jobs in parallel. The material places emphasis on the R programming language and parallelization of jobs using the mclapply function from the parallel package as well as GNU parallel.

It is likely that most of your questions can be answered while browsing the official project wiki, so don't forget to visit it.
